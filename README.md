# Build Instructions

## Requirements

- [pandoc](http://pandoc.org)
- LaTeX for PDF
- [ImageMagick](http://www.imagemagick.org/script/index.php) for HTML

## Build

To make a PDF:

```
make pdf
```

To make HTML:

```
make html
```
